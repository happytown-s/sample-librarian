sample-librarian
MCP Server
Python

An MCP server that searches, analyzes, and recommends local audio samples — then sends them to Ableton Live when needed.


Built on SQLite + FTS5 with BM25 full-text search, AI-friendly result enrichment, and harmonic compatibility scoring via the Camelot Wheel.


Works standalone — Ableton integration is optional.

🔍 SQLite + FTS5 BM25 search
🎵 BPM / key / pitch detection
🔁 Duplicate detection (4 axes)
🎹 Camelot Wheel recommendations
🤖 AI-enriched search results

live-agent-remote
MCP Server
Ableton Live

A standalone MCP server that gives AI agents full control of Ableton Live — create tracks, write MIDI, load samples, set warp markers, and build entire sessions.


Built on the LiveAgent Control Surface script, communicating via TCP socket. No Max for Live required.


Features undo groups, dry-run mode for safe previews, and an eval/exec safe mode (disabled by default, opt-in via env var).

🎹 Create MIDI / audio tracks
📝 Write MIDI notes to clips
🔊 Import & warp audio
🥁 Drum Rack management
↩️ Undo group support
🔒 dry_run safe preview

Get Started

2 servers
sample-librarian
$ git clone https://github.com/happytown-s/sample-librarian.git
$ cd sample-librarian && pip install -e .

# Add to your MCP client config:
"mcpServers": {
  "sample-librarian": {
    "command": "python",
    "args": ["mcp_server.py"]
  }
}
live-agent-remote
$ git clone https://github.com/happytown-s/live-agent-remote.git
$ cd live-agent-remote && pip install -e .

# Requires LiveAgent Control Surface script in Ableton Live
"mcpServers": {
  "liveagent": {
    "command": "python",
    "args": ["mcp_server.py"]
  }
}
